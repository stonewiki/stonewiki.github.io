如何实现分页功能

1.代码如下

$total = mysql_num_rows($res1);             
//页面大小               
$pagesize = 3;           
//页面的值           
$page = isset($_GET['page']) ?$_GET['page'] : 1;                
//页面最大数目                  
$maxpage = ceil($total/$pagesize);             
//偏移量             
$offset = ($page-1)*$pagesize;               
$sql = "select * from mess_info limit {$offset},{$pagesize}";
